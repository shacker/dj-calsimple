CalCentral: Pilot Release

Welcome to the all-new pilot release of CalCentral. As we improve this site, you may notice changes to some existing features, or new capabilities being added. This is all part of the development process, so stay tuned.

In this release, students, staff and faculty can:

  • Quickly access upcoming events in bCal
  • Find classes and groups in bSpace and Canvas
  • Keep tabs on classroom assignments
  • View your Google task list
  • Stay informed with campus announcements
  • Browse class listings in UC Berkeley departments
